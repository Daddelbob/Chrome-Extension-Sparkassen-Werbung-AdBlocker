Datenschutzerklärung - Chrome-Erweiterung „Sparkasse Werbung AdBlocker“

Stand: 12.08.2025

1. Verantwortlich

Herausgeber/in: Privatprojekt „Sparkasse Werbung AdBlocker“
Kontakt: severin.klug@gmail.com

2. Zweck der Erweiterung

Die Erweiterung entfernt auf ausgewählten Sparkassen-Webseiten ausschließlich interne Werbeelemente (z. B. Elemente mit der CSS-Klasse .cbox-campaign) durch eine lokale DOM-Manipulation im Browser der Nutzerin/des Nutzers.

3. Verarbeitete Daten

Es werden keine personenbezogenen Daten erhoben, gespeichert, analysiert oder an Dritte übermittelt.

  • Keine eigenen Netzwerkabfragen der Erweiterung
  • Keine Nutzung von chrome.storage (kein Speichern von Daten)
  • Keine Cookies, kein Tracking, keine Analysedienste

4. Berechtigungen

Die Erweiterung läuft nur auf den im Manifest angegebenen Hostmustern (Sparkassen-Domains). Die Ausführung dient allein dem Entfernen genannter Werbeelemente. Es werden keine weitergehenden Berechtigungen angefordert.

5. Weitergabe an Dritte

Es erfolgt keine Weitergabe von Daten an Dritte.

6. Aufbewahrung & Löschung

Es werden keine Daten gespeichert; eine Löschung ist daher nicht erforderlich.

7. Änderungen dieser Erklärung

Bei funktionalen Änderungen der Erweiterung wird diese Erklärung aktualisiert. Die jeweils aktuelle Fassung ist stets unter dieser URL abrufbar.

Hinweis: Diese Vorlage stellt keine Rechtsberatung dar. Bitte prüfe die Inhalte für deinen konkreten Anwendungsfall.
